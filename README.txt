Para a correta execução do trabalho seguir os seguintes passos:

1- Executar o "ConversorCSVParaBinario.cpp", este realizará a conversão do
arquivo "call911_1.csv" para um arquivo binário.

2- Aguardar o término da execução.

3- Após o término da execução do programa anterior, executar o 
"ManipulaArquivoBinario.cpp" para realizar operações desejadas
e a ordenação no arquivo convertido já para binário.
